### About

I'm an AI Engineer who builds intelligent *systems*, not just models. My work sits at
the intersection of machine learning and backend engineering — designing the pipelines,
APIs, and data layers that let a model actually function inside a real product.

I care less about a notebook that scores well and more about software that ships:
authenticated, documented, and built to handle real traffic and real users. That means
thinking about architecture as much as accuracy — where the data lives, how the model
is served, how the system behaves under load.

My work spans artificial intelligence, deep learning, computer vision, and NLP, through
to recommendation systems, generative AI, and LLM-based applications built with RAG and
prompt engineering — all backed by backend development that makes the intelligence
usable. Right now, I'm building **Glucora**, an AI-powered diabetes management app,
while sharpening my backend fundamentals through hands-on API projects.
